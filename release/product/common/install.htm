Инсталляция сервера Millennium Business Suite Anywhere 2007

Системные требования

Для функционирования сервера Millennium Business Suite Anywhere 2007 необходимо наличие одной из следующих операционных систем:

  • Linux
  • Solaris
  • Windows XP SP2
  • Windows Server 2003 SP1
  • Windows Vista

Инсталляция платформы Java

До инсталляции сервера Millennium Business Suite Anywhere 2007 на компьютер необходимо установить Java SDK версии не ниже 5.0.

Далее приведен пример для установки на операционную систему Microsoft Windows, за более подробной информацией об установке платформы Java SDK необходимо обратиться на http://java.sun.com/j2se/1.5.0/install.html

Запустите инсталлятор Java SDK (например, jdk-1_5_0_09-windows-i586-p.exe) и установите его в каталог, не содержащий в пути пробелов (например, C:\j2sdk1.5.0_09).

Установите переменную окружения JAVA_HOME в значение, соответствующее пути к установленному Java SDK. Сделать это можно следующим образом:

    1. Запустить ярлык панели управления "Система".
    2. На закладке "Дополнительно" нажать кнопку "Переменные среды".
    3. В списке "Системные переменные" нажать кнопку "Создать".
    4. Ввести имя переменной: JAVA_HOME и значение: C:\j2sdk1.5.0_09

Чтобы убедиться в том, что среда Java успешно установлена, откройте командную строку и выполните команду: java -version
В ответ команда выдаст информацию о версии установленной среды Java.

Запуск инсталлятора сервера Millennium Business Suite Anywhere 2007

Запустить инсталлятор сервера MillenniumERP-server-4.x.x.jar можно двумя способами:

    1. Из проводника двойным щелчком на файле инсталлятора.
    2. Из командной строки, с помощью команды: javaw -jar MillenniumERP-server-4.x.x.jar

Рекомендуется установить все компоненты, предложенные инсталлятором, в путь, не содержащий пробелов.

После инсталляции используйте файле readme.htm для получения инструкции по запуску установленного сервера Millennium Business Suite Anywhere 2007.
