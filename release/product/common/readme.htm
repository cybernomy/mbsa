Сервер Millennium Business Suite Anywhere 2007

Предварительные условия

Предполагается, что:

  1. На компьютере сервера приложения установлена платформа Java SDK (JDK) версии не ниже 1.5, переменная среды JAVA_HOME содержит путь к каталогу JDK.
  2. На компьютере используемом в качестве клиента установлена платформа Java (JRE) версии не ниже 1.5 и произведена интеграция плагинов Java с браузером. Если данное условие не выполнено, то браузер поддерживающий автоматическую установку плагинов предложит загрузить и установить платформу Java, необходимо следовать инструкциям браузера.
  3. На компьютере сервера базы данных установлена СУБД Firebird версии не ниже 2.0
  4. Все компоненты сервера Millennium Business Suite Anywhere 2007 установлены. Далее будем называть каталог установки системы MBSA_HOME.
  5. Сервер будет работать с локальной базой данных MBSA_HOME/appserver/server/merp/data/firebird/MERP4.FDB

Запуск сервера Millennium Business Suite Anywhere 2007

Для запуска сервера приложения необходимо перейти в каталог MBSA_HOME/appserver. Запуск сервера осуществляется с помощью команды StartServer.bat для платформы Windows или ./startserver.sh для платформы Linux.

Остановка сервера осуществляется либо с помощью команды StopServer.bat для платформы Windows или ./stopserver.bat для платформы Linux, либо нажатием комбинации клавиш Ctrl+C в окне запущенного сервера.

 

Подключение к системе Millennium Business Suite Anywhere 2007

Приложение будет запущено используя JNLP (Java Web Start) и подключено к серверу приложения по адресу $HOST_NAME: Стартовать используя JNLP (Java Web Start)

Приложение будет запущено используя Java аплет и подключено к серверу приложения по адресу $HOST_NAME: Стартовать используя аплет

Если на клиентском компьютере установлено JRE 6.0 и выше, то возможно использование приложения с внешним видом от substance. Для запуска необходимо использовать следующие ссылки.

Приложение будет запущено используя JNLP (Java Web Start) и подключено к серверу приложения по адресу $HOST_NAME: Стартовать используя JNLP (Java Web Start) с L&F Substance

Приложение будет запущено используя Java аплет и подключено к серверу приложения по адресу $HOST_NAME: Стартовать используя аплет с L&F Substance

При первом запуске приложения необходимо использовать следующие параметры подключения: пользователь admin, пароль admin.
